# Code of Conduct

We want termloop to be a welcoming project for contributors and users.

## Expected behavior

- Be respectful and constructive.
- Assume good intent.
- Keep feedback focused on the work, not the person.

## Unacceptable behavior

- Harassment, discrimination, or abusive language.
- Deliberate disruption or spam.
- Publishing others' private information without consent.

## Enforcement

Project maintainers may remove content or restrict participation when behavior violates this code of conduct.
