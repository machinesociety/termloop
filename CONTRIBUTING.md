# Contributing

Thanks for helping improve termloop.

## Development flow

1. Keep changes small and focused.
2. Update docs when behavior changes.
3. Prefer additive changes for providers and routing rules.

## Local setup

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -e .[dev]
```

## Pull requests

- Include a short summary of what changed.
- Mention any configuration or environment variables that new users need.
- Keep the OpenAI-compatible API stable unless a breaking change is intentional.
